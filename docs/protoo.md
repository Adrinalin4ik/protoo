## `protoo` module API

The entry module.

```javascript
var protoo = require('protoo');
```



### `protoo()` function

Creates an [Application](Application.md) instance.

```javascript
var app = protoo();
```


### Properties


#### `version`

Returns a String with the version of the library.

```javascript
protoo.version    // => "X.Y.Z"
```
